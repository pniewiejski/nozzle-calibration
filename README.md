# nozzle-calibration
School project that aims to find a method of identifying fuel nozzles with miscalibrated gauges. Work is based on simulated data sets.

## Prerequisites

### Data
Download data files. 
```
$ cd data
$ ./downloadData.sh [Google Drive id]
```

### Python dependencies
```
make install
```
